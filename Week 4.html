Exercise 5 - Fusion 360 Lego Brick

Building on the fusion exercises done last week, the exercises got a little more difficult. To start off this week, we had to make a lego brick in Fusion 360. Below show the specific dimensions used in this exercise.

First, I drew out a rectangle follwing the dimensions given, which was 31.80mm by 15.80mm

Then, using the extrude function, I created a block like this.

Next, I drew out the circles using sketch.

Similarly, these circles were then made into 3d using the extrude function. This forms the basic shape of the lego block, however, i was not done yet.

Turning to the underside of the block, I had to ensure that it was not just a solid block so i clicked the shell function to create what is seen in the image below.

After that, I still had to continue with the inside of the lego , this is where they are able to click with one another. I again used the sketch function to form the circles and then used the offset button to create the smaller circles as seen.

Again , like the other two times , I extruded the spaces between the two circles to form this.

Finally, to finish off this brick, i used the fillet function to smoothen the corners of the lego brick so that they were not too sharp. And this is the final outcome !!

Exercise 6 - Fusion 360 Name Tag

The next exercise of this week is this name tag . The dimensions are shown in the image below.

To start off, I created this with the sketch function. The bigger part was made by using a rectangle and two circles at the sides, then using the trim function to combine it . After that the smaller circle was added separately. To achieve the result seen in the screenshot, the offset function was used.

Then , I just simply used the extrude button for this.

I then redrew the offset part on the extrude part to continue with the next step.

Again, I then extruded the sketch from before.

After so , the name was added in , or in this case the label of FabLab.

Of course, then the extrude button was used again to produce this final result.

Exercise 8 - Fusion 360 Mug with Handle

Okay for the final exercise done this week, it is a mug with a handle. This one was a little more tricky compared to the other three done. Like the ones above, the dimensions are stated in the image below.

To start of , a section of the mug was drawn . Or more specifically, half of a section was drawn. I used sketch to draw this out again.The line and arc tool was used here to create this section.

So here, this time instead of the extrude tool, I used the revolve tool to form this sort of cylinder shape as seen.

Next, I created the handle by sketching with lines and arcs again. The offset tool was also used here.

Back to the extrude tool again, the handle of the mug was formed.

Finally , I formed the final mug by combining the handle part with the cup part of the mug. This is the final result.

The mug as done in fusion 360.
